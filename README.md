# Hola, soy Javi Tatay 👋

**Técnico audiovisual · Docente · Maker de herramientas AV**

Trabajo en producción técnica de espectáculos en vivo — sonido, iluminación, vídeo y RF — y enseño ciclos formativos de audiovisuales en Valencia. Lo que no encuentro lo construyo: herramientas offline, single-file, sin dependencias, hechas para el sector AV.

---

## 🛠️ Proyectos

| Proyecto | Qué hace |
|---|---|
| [**RFTDT**](https://github.com/javitatay/RFTDT) | Consulta de frecuencias UHF-TDT por demarcación para técnicos de RF. Espectro visual, verificador de conflictos y calculadora de intermodulación IM3 |
| [**VectorSlice**](https://github.com/javitatay/VectorSlice) | Convierte vectores de Illustrator en Slices de Resolume Arena — vía web o script .jsx nativo, sin redibujado |
| [**AVV**](https://github.com/javitatay/AVV) | Recursos y referencias para animación visual en vivo |
| [**Tatimer**](https://github.com/javitatay/Tatimer) | Timer minimalista para eventos en directo |
| [**Tarimeo**](https://github.com/javitatay/Tarimeo) | App de gestión de tarimas en espectáculos |
| [**QLab Scripts**](https://github.com/javitatay/QLab-Scripts-Live-Events) | Scripts AppleScript para integrar apps externas en shows desde QLab 5 |
